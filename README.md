# Тестовое задание для Modesto Flowers
## Краткая инструкция по работе
### Для начала работы у вас должент быть установлен:
* Node.js v18.12.1.
* Gulp v4
* npm last version
## Основные команды для работы
* Установка - `npm i`
* Сборка проекта и запуск локального сервера - `gulp`
